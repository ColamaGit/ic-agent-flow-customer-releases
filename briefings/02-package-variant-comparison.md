# Package Variant Comparison

| Variant | Target User | Scope | Governance |
|---|---|---|---|
| `solo_eval` | single evaluator | minimal baseline | non-production |
| `team_project` | small team | team collaboration + RBAC | team-governed |
| `enterprise_site` | organization deployment | approval/audit governance | enterprise-governed |

## Rule

`solo_eval` success must not be used to claim `enterprise_site` readiness.
