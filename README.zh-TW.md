# ic-agent-flow Customer Releases（繁體中文）

[EN](./README.md) | [繁中](./README.zh-TW.md)

此倉庫是 `ic-agent-flow` 的 customer-facing release registry（對客交付面），不是開發 source repo。

## 你可以在這裡找到什麼

- 發佈索引：`RELEASE_INDEX.md`
- 版本差異：`releases/CHANGELOG.md`
- 交付包：`releases/runtime-bundle-tw-*`
- 版本指標：`releases/version-pointers/*.json`（含 `LATEST.json`）
- 對客操作文件：`docs/`
- 客戶/來賓閱讀包：`briefings/`

## i18n 說明

- 本 repo 採 EN / zh-TW 雙語支援。
- 對應規則請見：`docs/i18n-policy.md`。
- `*.zh-TW.md` 若標示 fallback，代表先提供導覽版，後續會逐步升級完整對譯。
